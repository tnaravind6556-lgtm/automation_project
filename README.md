# Make.com Google Sheets Automation

This project is a Make.com automation blueprint designed to integrate Google Sheets with AI powered processing using Google Gemini. 
It automates repetitive tasks such as data handling, transformation, and workflow execution, reducing manual effort and improving efficiency.
The automation is built to be modular, scalable, and easy to customize for different real world use cases such as data enrichment, reporting, and process automation.

* What problem does this automation solve ? 

Many workflows involving Google Sheets require repetitive actions like copy-pasting, manual formatting, decision making, or content generation. 
This automation eliminates those repetitive tasks by using AI to automatically process, analyze, and respond to data changes.

* What tools and technology are used 

Make.com – Workflow automation platform.
Google Sheets – Data source and storage.
Google Gemini AI – AI-powered processing.
APIs – For system to system communication.

* Why did i build this? 

This project was built as a learning and portfolio initiative to understand how AI can be integrated into real world automation workflows. 
It demonstrates how repetitive tasks can be replaced with intelligent systems that save time and improve efficiency.
